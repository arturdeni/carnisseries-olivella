# Locomotive Smooth Scroll Demo

Proyecto de demostración usando Locomotive Scroll para crear efectos de scroll suave y parallax.

## Estructura

```
├── index.html
├── style.css
├── script.js
├── assets/
│   └── images/
└── README.md
```

## Tecnologías

- [Locomotive Scroll](https://github.com/locomotivemtl/locomotive-scroll) - Smooth scroll library
- [Lenis](https://github.com/darkroomengineering/lenis) - Smooth scrolling CSS

## Uso

1. Abre `index.html` en tu navegador
2. Añade tus imágenes AVIF en `assets/images/`

## Personalización

- Modifica `style.css` para cambiar colores y tipografías
- Ajusta `multiplier` en `script.js` para cambiar la velocidad del scroll
- Usa `data-scroll-speed` en elementos para efectos parallax
